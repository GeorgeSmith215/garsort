# 环境初始化

- 安装python3

- 安装各种库和依赖：

- ```python
  pip3 install flask
  pip3 install tensorflow==1.13.1
  pip3 install keras==2.3.1
  pip3 install opencv
  pip3 install requests
  pip3 install Pillow
  ```

  
# 运行

- 命令`python3 run.py`开启窗口本地调试
- 命令`python3 flask_sever.py`开启服务部署
- 命令`sh ./start.sh`开启后台运行服务部署
